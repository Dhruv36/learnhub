Autoplay Reality

<video autoplay muted loop playsinline src="/media/hero-bg.mp4"></video>

Browsers block autoplay with sound. The only autoplay that works everywhere is muted (+ playsinline for iOS). That's why every hero-section background video you've ever seen is silent.

Embedding Third-Party Content (iframes)

<iframe src="https://www.youtube-nocookie.com/embed/VIDEO_ID"
        title="Launch keynote"
        loading="lazy" allowfullscreen
        sandbox="allow-scripts allow-same-origin allow-presentation"></iframe>

title for accessibility, loading="lazy" for performance, sandbox to limit what the embedded page can do — the same mechanism this site's Playground uses to run your code safely.

💡 Production tip: at real scale nobody serves one MP4 file — video is transcoded into adaptive bitrate streams (HLS/DASH) so quality adjusts to bandwidth. The <video> element stays the same; a library (hls.js) feeds it segments. Full pipeline in the System Design track's media lessons.
