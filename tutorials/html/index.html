HTML Introduction Basics

HTML (HyperText Markup Language) is the standard language for building web pages. Every website you use in production — from Google to your banking app — starts with HTML.

Your First Page

<!DOCTYPE html>
<html>
  <head><title>My Page</title></head>
  <body><h1>Hello, World!</h1></body>
</html>
💡 Production tip: Real products always declare <!DOCTYPE html> and lang — browsers and screen readers depend on them.
