SEO & Meta Tags Advanced

Search engines read your HTML the way screen readers do: structure first. Everything from the Semantic HTML lesson doubles as SEO. This lesson adds the metadata layer that controls how your pages appear in search results and social shares.

The Head of a Well-Ranked Page

<title>Standing Desk DX-1 — Ergonomic Motorized Desk | DeskCo</title>
<meta name="description" content="Height-adjustable desk with dual motors,
  memory presets, and a 10-year warranty. Free shipping.">
<link rel="canonical" href="https://deskco.com/desks/dx-1">
<meta name="robots" content="index, follow">
TagWhat it controls
title (≤ 60 chars)The blue link in results — the single strongest on-page signal
description (≤ 155 chars)The snippet below it — controls click-through rate, not ranking
canonicalWhich URL is "the real one" when content is reachable at several (with/without ?utm=…) — prevents duplicate-content dilution
robotsnoindex keeps staging sites and thin pages out of Google

Social Cards (Open Graph + Twitter)

<meta property="og:title" content="Standing Desk DX-1">
<meta property="og:description" content="Dual motors, memory presets.">
<meta property="og:image" content="https://deskco.com/img/dx1-card.jpg">  <!-- 1200×630 -->
<meta property="og:url" content="https://deskco.com/desks/dx-1">
<meta name="twitter:card" content="summary_large_image">

These decide the preview card when your link is pasted into Slack, LinkedIn, X, iMessage. No og:image = gray placeholder = dramatically fewer clicks.

Structured Data (Rich Results)

<script type="application/ld+json">
{ "@context": "https://schema.org", "@type": "Product",
  "name": "Standing Desk DX-1",
  "offers": { "@type": "Offer", "price": "499", "priceCurrency": "USD" },
  "aggregateRating": { "@type": "AggregateRating", "ratingValue": "4.7", "reviewCount": "1284" } }
</script>

This JSON-LD is what produces the ⭐ 4.7 stars, price, and availability directly in Google results. Recipe sites, job boards, and e-commerce live and die by it.

💡 Production tip: the highest-ROI SEO work is rarely meta tags — it's one <h1> per page, real <a href> links (crawlers don't click JS divs), fast Core Web Vitals (next lesson), and canonical discipline. Validate with Google's Rich Results Test before shipping.
