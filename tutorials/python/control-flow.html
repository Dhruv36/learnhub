Control Flow & Functions Basics

Python's control flow reads like pseudocode, but functions hide two of the most-tested gotchas in the language: the mutable default argument and the difference between positional and keyword arguments. This lesson covers the flow constructs and the function model senior interviews probe.

Control Flow

if score >= 90: grade = "A"
elif score >= 80: grade = "B"
else: grade = "C"

for i, item in enumerate(items, start=1):     # index + value, Pythonic
    if item.skip: continue
    if item.stop: break
else:                                          # runs if the loop did NOT break (rare but real)
    all_processed()

for a, b in zip(names, scores): ...            # parallel iteration
while queue:                                   # truthiness: empty collection is falsy
    task = queue.pop()

# match (structural pattern matching, 3.10+) — more than a switch:
match command.split():
    case ["go", direction]:      move(direction)
    case ["quit" | "exit"]:      shutdown()
    case ["add", *args]:         add(args)
    case _:                      unknown()
🐍 The for-else and while-else clauses run only when the loop finishes without break — handy for search loops ("else: not found"). Obscure but a favorite interview trivia question.

THE Mutable Default Argument Trap

def add_item(item, cart=[]):        # ❌ the default list is created ONCE, at def time
    cart.append(item)
    return cart
add_item("a")    # ['a']
add_item("b")    # ['a', 'b']  😱 — SAME list persists across calls!

def add_item(item, cart=None):      # ✅ the sentinel pattern
    if cart is None: cart = []
    cart.append(item)
    return cart
# Default values are evaluated ONCE when the function is defined, not per call.
# So a mutable default is shared state. This is the #1 Python function interview question.

Return Values, Scope & Closures

def stats(nums):
    return min(nums), max(nums), sum(nums)/len(nums)   # returns a TUPLE
lo, hi, avg = stats(data)                               # unpacked

count = 0
def inc():
    global count; count += 1        # rebind a module-level name (avoid; prefer returns)
def make_counter():
    n = 0
    def step():
        nonlocal n; n += 1          # rebind the ENCLOSING function's name (closures)
        return n
    return step
# LEGB scope resolution: Local → Enclosing → Global → Built-in. Reading walks outward;
# WRITING to an outer name needs global/nonlocal (else Python makes a new local).

Interview Questions

Q1: Why does a mutable default argument persist across calls?

Default values are evaluated once, when the def statement executes — not on each call. A default list/dict is therefore a single shared object living on the function; mutating it accumulates state across calls. The fix is the None sentinel: default to None and create a fresh object inside the body. It's the classic "explain this output" screening question.

Q2: Explain LEGB and why writing to an outer variable behaves surprisingly.

Name lookup order: Local, Enclosing (nested functions), Global (module), Built-in. Reading walks outward. But an assignment anywhere in a function makes that name local for the WHOLE function — so reading it before the assignment raises UnboundLocalError, and you can't rebind an outer name without global (module) or nonlocal (enclosing). Prefer returning values over rebinding outer state.

Q3: What do / and * do in a function signature?

They partition parameters: everything before / is positional-only (can't be passed by keyword), everything after * is keyword-only (can't be passed positionally). They let API authors keep positional args as an unstable implementation detail (rename freely) while forcing clarity for optional flags (must be named) — the readability discipline from the JS functions lesson, enforced by the language.

🏋️ Practice: reproduce the mutable-default bug and fix it with the sentinel; write a function returning (min,max,avg) and unpack it; use enumerate+zip to build a numbered leaderboard; write a make_counter closure with nonlocal; use match to parse simple commands.

🎯 Key Takeaways

  • enumerate/zip/direct iteration over range(len(...)); for-else runs when no break occurred.
  • Argument model: positional, defaults, *args, keyword-only, **kwargs; / and * control call style.
  • Mutable default args are evaluated once and shared — use the None sentinel (the #1 gotcha).
  • LEGB scope; assignment makes a name local for the whole function; global/nonlocal to rebind outer.
  • match is structural pattern matching (destructuring), not just a switch; catch Exception, not bare except.
