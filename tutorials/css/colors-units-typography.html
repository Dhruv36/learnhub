Colors, Units & Typography Basics

Colors — Four Syntaxes, One Rule

color: #2f8d46;                    /* hex — most common in codebases */
color: rgb(47 141 70 / 0.8);       /* rgb + alpha */
color: hsl(140 50% 37%);           /* hue/saturation/lightness — easy to tweak */
color: oklch(0.55 0.15 150);       /* perceptually uniform — the modern pick */

The rule: never hardcode colors at point of use — define them once as custom properties (--brand: #2f8d46) and reference everywhere. Full pattern in the Theming lesson.

Units — When to Use Which

UnitRelative toUse for
pxFixedBorders, shadows, tiny details
remRoot font size (16px default)Font sizes, spacing — the default choice
emParent font sizePadding that scales with the element's own text (buttons)
%Parent dimensionFluid widths
vw/vh/dvhViewportFull-screen sections (dvh handles mobile URL bars)
chWidth of "0"Readable line lengths: max-width: 65ch

Why rem over px for text: users who raise their browser's default font size (accessibility setting) get larger text with rem — px ignores them. WCAG audits check this.

Typography That Reads Well

body {
  font-family: system-ui, -apple-system, "Segoe UI", Roboto, sans-serif;
  font-size: 1rem;          /* 16px baseline — never smaller for body text */
  line-height: 1.6;         /* unitless multiplier — inherits correctly */
}
h1 { font-size: clamp(1.8rem, 4vw, 2.5rem); }   /* fluid: min, preferred, max */
p  { max-width: 65ch; }                          /* 45–75 chars = readable line */
PropertyGuidance
line-height1.5–1.7 body, 1.1–1.3 headings; always unitless
font-weight400 body / 600–700 headings; more than 3 weights = slow font loads
letter-spacingSlightly negative for big headings (-0.02em), never negative for body
clamp()Fluid type without media queries — the modern heading pattern
💡 Production tip: the system font stack above costs 0 bytes and 0ms — it uses each OS's native UI font. Ship a webfont only when brand demands it, and revisit the HTML track's Performance lesson (preload + font-display: swap) when you do.
