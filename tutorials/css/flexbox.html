Flexbox Intermediate

Flexbox lays out items in one dimension — a row or a column — and excels at distributing space and aligning things. Navbars, toolbars, card innards, form rows: flexbox. Two-dimensional page layouts: that's Grid (next lesson).

The Mental Model: Main Axis vs Cross Axis

.container { display: flex; flex-direction: row; }     /* main axis → horizontal */

 main axis (row) ────────────────────────▶
┌────────┬────────┬────────┐   ▲
│ item 1 │ item 2 │ item 3 │   │ cross axis
└────────┴────────┴────────┘   ▼

justify-content → positions along MAIN axis
align-items     → positions along CROSS axis
(flip direction to column and they swap meaning)

Container Properties (the 90% set)

.toolbar {
  display: flex;
  justify-content: space-between;  /* start | center | space-between | space-around */
  align-items: center;             /* stretch | start | center | baseline */
  gap: 1rem;                       /* space BETWEEN items — use this, not margins */
  flex-wrap: wrap;                 /* allow items onto new lines */
}

Item Properties — How Items Grow & Shrink

flex: 1;            /* = flex-grow:1, shrink:1, basis:0 — share space equally */
flex: 0 0 240px;    /* rigid 240px: don't grow, don't shrink (sidebars) */
flex: 2;            /* twice the share of a flex:1 sibling */
margin-left: auto;  /* the famous trick: pushes THIS item (and rest) to the far end */
align-self: center; /* override align-items for one item */

Three Layouts You'll Build This Week

/* 1. Navbar: logo left, links right */
.nav { display: flex; align-items: center; gap: 1rem; }
.nav .links { margin-left: auto; display: flex; gap: 1rem; }

/* 2. Media object: avatar + text that fills */
.row { display: flex; gap: .8rem; }
.row .body { flex: 1; min-width: 0; }    /* min-width:0 lets long text truncate */

/* 3. Sticky footer page */
body { min-height: 100dvh; display: flex; flex-direction: column; }
main { flex: 1; }                         /* footer sits at bottom even on short pages */

The Bug Everyone Hits: min-width: auto

Flex items refuse to shrink below their content's size by default — one long unbreakable string (a URL, a long word) blows the layout open. Fix: min-width: 0 on the item (plus overflow: hidden; text-overflow: ellipsis; white-space: nowrap to truncate prettily).

💡 Production tip: reach for gap instead of margins between flex children every time — no first/last-child exceptions, no margin collapsing, works in wrap. DevTools' flex badge (click it in the Elements panel) visualizes the axes live.
