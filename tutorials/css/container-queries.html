Container Queries Advanced

Media queries answer "how wide is the viewport?" — but a component doesn't care about the viewport. The same ProductCard renders in a wide main column AND a narrow sidebar on the same page. Container queries let components respond to their own available space — the missing piece that makes design-system components truly reusable.

The Two Steps

/* 1. Name a containment context on the PARENT: */
.card-slot { container: card / inline-size; }     /* name / what to measure */

/* 2. Query it from the children: */
.product-card { display: flex; flex-direction: column; }   /* narrow default */

@container card (min-width: 420px) {
  .product-card { flex-direction: row; }           /* wide: image beside text */
  .product-card img { width: 40%; }
}

Now the card is horizontal in the main column, vertical in the sidebar — same component, zero props, zero JS, no viewport math.

Container Query Units

@container card (min-width: 300px) {
  .product-card h3 { font-size: clamp(1rem, 5cqi, 1.5rem); }
}
/* cqi = 1% of the container's inline size (also cqw, cqh, cqb, cqmin, cqmax)
   Fluid type relative to the COMPONENT, not the screen. */

Style Queries (the frontier)

.theme-dark { --theme: dark; }
@container style(--theme: dark) {
  .card { background: #161b22; }
}
/* Query a parent's custom property value — condition components on CONTEXT, not just size. */

When Media Queries Still Win

Use container queries for…Use media queries for…
Component-level adaptation (cards, widgets, tables)Page-level layout (sidebar visible? nav collapsed?)
Anything that ships in a design systemUser preferences: dark mode, reduced motion, print
💡 Production tip: inline-size containment means the container's width can't be dictated by its children (that's what makes measuring it non-circular) — so the parent must get its width from layout (grid/flex/explicit). If a container query "isn't firing", the container almost always has no constrained width. Browser support has been green across the board since 2023 — this is production-safe today.
