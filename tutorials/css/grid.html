CSS Grid Intermediate

Grid lays out in two dimensions — rows AND columns at once. Page shells, dashboards, photo galleries, card grids: Grid. It also ends the float-era hacks for good.

The One-Liner That Replaced a Framework

.cards {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(220px, 1fr));
  gap: 1rem;
}
/* As many 220px+ columns as fit, sharing leftover space, wrapping automatically.
   Responsive with ZERO media queries. (This site's homepage cards = exactly this.) */

Explicit Grids

.layout {
  display: grid;
  grid-template-columns: 240px 1fr 300px;   /* sidebar | content | panel */
  grid-template-rows: 60px 1fr 40px;        /* header | body | footer */
  gap: 0 1rem;
}
/* fr = fraction of free space. minmax(0,1fr) prevents content blowouts. */

Placing Items — Lines & Areas

/* By line numbers: */
.header { grid-column: 1 / -1; }          /* span from first to last line */
.hero   { grid-column: 2 / 4; grid-row: 1 / 3; }

/* Or the readable way — named areas: */
.layout {
  grid-template-areas:
    "header header header"
    "nav    main   aside"
    "footer footer footer";
}
.header { grid-area: header; }  .nav { grid-area: nav; }  /* … */

Areas make the layout visible in the CSS itself — and rearranging for mobile is just redefining the map inside a media query.

Grid vs Flexbox — the Interview Question

FlexboxGrid
DimensionsOne (row OR column)Two (rows AND columns)
Control lives in…The items (grow/shrink)The container (template)
Best forContent-out: toolbars, nav, unknown item counts in a lineLayout-in: page shells, dashboards, galleries

They compose: Grid for the page, Flexbox inside each cell. Not rivals — layers.

Modern Bonus: subgrid

.card { display: grid; grid-template-rows: subgrid; grid-row: span 3; }
/* Cards in a row share row tracks — titles align, buttons align, regardless of content length. */
💡 Production tip: DevTools' grid inspector (click the "grid" badge) overlays line numbers and area names on the live page — build layouts with it open. And prefer auto-fill + minmax() before writing any column media query; most "responsive grid" code in the wild is obsolete.
